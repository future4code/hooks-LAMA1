> 👨‍💻 Como rodar o projeto 
  
* Clonar o repositório
  
* Rodar `npm install`
  
* Criar um arquivo `.env` na pasta raiz do projeto:

* Documentação no arquivo request.rest
  
 ``` 
   PORT = 3003
   DB_HOST = 
   DB_USER = 
   DB_PASSWORD = 
   DB_DATABASE = 
```

* Rodar `npm start` para ligar o servidor
